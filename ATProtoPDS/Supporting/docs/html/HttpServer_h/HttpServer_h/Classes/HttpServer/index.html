Introduction

HTTP server for handling PDS requests.

Discussion

HttpServer provides a simple HTTP server implementation for the PDS. It supports route registration for different HTTP methods and paths, with handlers invoked for matching requests.

@code HttpServer *server = [HttpServer serverWithPort:8080];

[server addRoute:@"GET" path:@"/health" handler:^(HttpRequest *req, HttpResponse *resp) { resp.statusCode = 200; [resp setBody:@"OK"]; }];

[server startWithError:nil]; @endcode
