Introduction

Block storage methods for PDSDatabase.

Discussion

These methods manage CAR blocks stored in the repository. Blocks contain serialized content indexed by their CID.


Methods

-deleteBlock:repoDid:error:

Deletes a block from the database.

-getBlockCountForRepo:error:

Counts the total blocks in a repository.

-getBlocksForRepo:limit:offset:error:

Retrieves blocks from a repository with pagination.

-getBlockWithCid:repoDid:error:

Retrieves a block by CID.

-saveBlock:error:

Saves a single block to the database.

-saveBlocks:error:

Saves multiple blocks in a batch operation.

deleteBlock:repoDid:error:

Deletes a block from the database. 

- (BOOL)deleteBlock:(NSData *)cid repoDid:(NSString *)repoDid 
        error:(NSError **)error;
Parameters
cid

The CID of the block to delete.

repoDid

The DID of the repository that owns the block.

error

On return, contains an error if the operation failed.

Return Value

YES if the block was deleted successfully, NO otherwise.
