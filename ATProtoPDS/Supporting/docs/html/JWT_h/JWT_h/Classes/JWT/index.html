JWT

Superclass:
NSObject
Declared In:
JWT.h

Introduction

Represents a complete JWT token.

Discussion

JWT provides methods for parsing, encoding, and accessing JWT tokens. It combines header, payload, and signature components.

@code // Parse a JWT token JWT *jwt = [JWT jwtWithToken:tokenString error:nil];

// Access claims NSString *issuer = jwt.payload.iss; NSString *subject = jwt.payload.sub;

// Get encoded form NSString *encoded = [jwt encodedToken]; @endcode
