JWTPayload

Superclass:
NSObject
Declared In:
JWT.h

Introduction

Represents the claims portion of a JWT.

Discussion

The JWT payload contains the claims about the token subject, including standard claims (iss, sub, aud, exp, iat, nbf) and ATProto-specific claims (did, handle, scope).
