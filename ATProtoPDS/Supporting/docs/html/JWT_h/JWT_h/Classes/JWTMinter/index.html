JWTMinter

Superclass:
NSObject
Declared In:
JWT.h

Introduction

Creates and signs JWT tokens.

Discussion

JWTMinter handles minting access tokens and refresh tokens with proper claims and signatures.


Methods

Signs a payload and returns a JWT string.

mintAccessTokenForDID:handle:scopes:error:

Mints an access token for a user. 

- (JWT *)mintAccessTokenForDID:(NSString *)did handle:(NSString *)handle 
        scopes:(NSArray<NSString *> *)scopes error:(NSError **)error;
Parameters
did

The user's DID.

handle

The user's handle.

scopes

The granted scopes.

error

On return, contains an error if minting failed.

Return Value

A new access token JWT.
