OAuth2AuthorizationRequest

Superclass:
NSObject
Declared In:
OAuth2.h

Introduction

Represents an OAuth 2.0 authorization request.

Discussion

This class encapsulates all parameters for an authorization request including client ID, redirect URI, scope, and PKCE parameters.
