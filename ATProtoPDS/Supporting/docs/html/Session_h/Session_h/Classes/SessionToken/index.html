SessionToken

Superclass:

Declared In:

Introduction

Represents an OAuth 2.0 token.

Discussion

SessionToken encapsulates both access tokens and refresh tokens with their associated metadata including expiration and scope.

isExpired

Checks if the token has expired. 

- (BOOL)isExpired;
Return Value

YES if the token has expired, NO otherwise.

Discussion

This method compares the current date against the expiration date. A small clock skew tolerance may be applied.

isValid

Checks if the token is currently valid. 

- (BOOL)isValid;
Return Value

YES if the token is valid, NO otherwise.

Discussion

A token is valid if it has not expired and has a valid format.
