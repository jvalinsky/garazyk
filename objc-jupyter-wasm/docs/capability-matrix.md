## Reproduction

From the repository root, run:

```bash
PLAYWRIGHT_CHROMIUM_EXECUTABLE_PATH=/path/to/chromium \
  bash objc-jupyter-wasm/scripts/run-capability-baseline.sh
```

The browser path is optional. Without it, the matrix records the browser row as not run while all
kernel, runtime, notebook, compatibility, and Nix asset checks still execute. The current browser
evidence used Chromium at `/Applications/Chromium.app/Contents/MacOS/Chromium` on macOS.
