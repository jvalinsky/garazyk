    Interface AgentProxy

    A dynamic proxy for XRPC method calls on an authenticated agent.

    Usage: await client.agent.createAccount({ handle, email, password }); await client.agent.login({ identifier, password }); await client.agent.com.atproto.repo.createRecord({ ... });

    The proxy builds method paths via property access and invokes them via function call. Returns { data } on success, throws on error.
