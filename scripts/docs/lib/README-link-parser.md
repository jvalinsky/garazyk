# Link Parser Module

This module documents `link-parser.js`, which parses markdown links for validation and migration tasks.

## Responsibilities

- Extract markdown links and image references.
- Identify anchors, local files, and external URLs.
- Normalize links before path resolution.

## Usage

```bash
node scripts/docs/lib/link-parser.js
```
