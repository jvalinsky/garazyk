Extends Class:
PDSDatabase
Declared In:
PDSDatabase.h

Introduction

Blob metadata methods for PDSDatabase.

Discussion

These methods manage blob references and metadata. Blobs are large binary attachments stored separately from the repository data.


Methods

-deleteBlob:error:

Deletes a blob from the database.

-getBlobCountForDid:error:

Counts blobs uploaded by an account.

-getBlobsForDid:limit:offset:error:

Retrieves blobs uploaded by an account.

-getBlobWithCid:error:

Retrieves blob metadata by CID.

-saveBlob:error:

Saves blob metadata to the database.

deleteBlob:error:

Deletes a blob from the database. 

- (BOOL)deleteBlob:(NSData *)cid error:(NSError **)error;
Parameters
cid

The CID of the blob to delete.

error

On return, contains an error if the operation failed.

Return Value

YES if the blob was deleted successfully, NO otherwise.

error

On return, contains an error if the operation failed.

getBlobsForDid:limit:offset:error:

Retrieves blobs uploaded by an account. 

- (NSArray<PDSDatabaseBlob *> *)getBlobsForDid:(NSString *)did 
        limit:(NSInteger)limit offset:(NSInteger)offset error:(NSError **)error;
Parameters
did

The DID of the account.

limit

Maximum number of blobs to return.

offset

Number of blobs to skip.

error

On return, contains an error if the operation failed.

Return Value

An array of blob objects.
