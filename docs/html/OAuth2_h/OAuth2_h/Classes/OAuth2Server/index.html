Introduction

OAuth 2.0 authorization server implementation.

Discussion

OAuth2Server handles all authorization server operations including authorization requests, token issuance, and token refresh. It integrates with JWT minting, key management, and identity resolution.

@code OAuth2Server *server = [[OAuth2Server alloc] init]; server.issuer = @"https://pds.example.com"; server.authorizationEndpoint = @"https://pds.example.com/oauth/authorize"; server.tokenEndpoint = @"https://pds.example.com/oauth/token";

[server handleAuthorizationRequest:request completion:^(URL, code, error) { // Redirect user to URL with code }]; @endcode
