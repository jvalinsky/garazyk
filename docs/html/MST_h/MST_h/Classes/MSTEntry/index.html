Introduction

Represents a key-value entry in the MST.

Discussion

MSTEntry stores a key and its associated content CID. For collections, a subKey can be used to distinguish multiple values within the same collection namespace.

@code // Create an entry for a record MSTEntry *entry = [MSTEntry entryWithKey:@"app.bsky.actor.profile" valueCID:recordCID]; @endcode
