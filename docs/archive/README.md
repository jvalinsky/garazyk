# Archived Documentation

This directory contains older documentation that is no longer actively maintained but preserved for historical reference.

## Subdirectories

- `planning/` - Completed or superseded planning documents
- `troubleshooting/` - Historical troubleshooting logs
- `reports/` - Older audit and analysis reports

## Notes

- Recent documentation is in the parent `docs/` directory
- Active planning notes are in root-level `plan_*.txt` files
- Current security and remediation reports are in `docs/security-and-architectural-remediation-report-2026-03-02.md`
