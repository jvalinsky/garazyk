# Historical Reports

This directory contains one-time session reports, audit artifacts, and phase checkpoints that have been superseded by canonical documentation. They are retained for historical reference only.

## Categories

- **Debug sessions**: `debug-session-*.md` — ephemeral debugging notes
- **Phase checkpoints**: `PHASE_*`, `APPVIEW_PHASE5_*`, `FINAL_*` — milestone summaries
- **Audit reports**: `*_REPORT.md`, `*-report-*.md` — one-time quality/security audits
- **Gap analyses**: `*-gap-analysis-*.md` — point-in-time coverage snapshots
- **Migration artifacts**: `migration-report.*`, `MIGRATION_*` — migration verification
- **Project summaries**: `SUMMARY.md`, `PROJECT_SUMMARY.md` — superseded by canonical docs
