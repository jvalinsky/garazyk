# Context Map

This repo is organized by service. Each service has its own domain language.

| Service | Source Directory | Description |
|---------|-----------------|-------------|
| PDS | `Garazyk/Sources/App/`, `Garazyk/Sources/Services/` | Personal Data Server — the main ATProto PDS implementation |
| AppView | `Garazyk/Sources/AppView/` | App View — indexed view of the network |
| Chat | `Garazyk/Sources/Chat/` | Chat service — DMs and conversation management |
| PLC | `Garazyk/Sources/PLC/` | PLC directory — DID-to-handle resolution |
| Relay | `Garazyk/Sources/Sync/`, `Garazyk/Sources/Relay/` | Relay — firehose aggregation and crawling |
| Video | `Garazyk/Sources/Video/` | Video processing — transcoding and thumbnail generation |
