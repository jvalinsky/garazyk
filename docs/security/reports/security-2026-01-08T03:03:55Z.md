### H-001: Deprecated API Usage
**Severity:** HIGH
**File:** `ATProtoPDS/Sources/App/PDSController.m:284`
**CWE:** CWE-477 (Obsolete API Usage)

**Description:**
Use of deprecated `NSURLConnection sendSynchronousRequest:` which was deprecated in macOS 10.11 and removed from newer SDKs.

**Code:**
```objc
NSData *responseData = [NSURLConnection sendSynchronousRequest:request
                                              returningResponse:&response
                                                          error:&requestError];
```

**Recommendation:**
Replace with `NSURLSession dataTaskWithRequest:completionHandler:` or use `sendAsync` variants.

### H-002: Missing Nullability Annotations
**Severity:** HIGH
**File:** `ATProtoPDS/Sources/Auth/Session.h:331`
**CWE:** CWE-476 (NULL Pointer Dereference)

**Description:**
Double pointer parameters lack nullability specifiers, preventing compiler from enforcing null checks.

**Code:**
```objc
- (BOOL)createSessionWithDID:(NSString *)did
                       scope:(NSString *)scope
                  dpopJWK:(NSDictionary *)dpopJWK
               newSession:(Session **)newSession  // Missing _Nullable/_Nonnull
                      error:(NSError **)error;
```

**Recommendation:**
Add nullability specifiers:
```objc
newSession:(Session * _Nullable * _Nonnull)newSession
```

**Status:** Unresolved
